Aiuto di Yapbam: I backup

I backup automatici

Il backup dei dati é essenziale per garantirne la sicurezza del tuo lavoro.

Yapbam ti permette di fare una copia compressa di ogni file di salvataggio, sia su un supporto di memorizzazione montato sul PC (chiave USB, disco interno o esterno), oppure via ftp, su un server che espone questo servizio.

Questa seconda soluzione é consigliata, soprattutto se questo server si trova in un luogo diverso dal computer normalmente utilizzato (in caso di incendio o inondazione, questo é l'unico modo sicuro per non compromettere i tuoi dati).

La maggior parte degli operatori Internet forniscono un server FTP.
Consulta il sito del tuo operatore o il tuo motore di ricerca preferito per saperne di più.
